## SQLite 3 Connection Page

You can use this type of SQL connection to run your scripts on an SQLite database.

![Sqlite connection](https://cdn.advancedinstaller.com/img/ui/sqlite-connection.png "Sqlite connection")  

![Note](https://cdn.advancedinstaller.com/svg/common/IconMessageNote.svg)There is no need to have an SQLite library to install (e.g. as a prerequisite). This .DLL is included automatically when a script is used.
